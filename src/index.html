**About This App**

Inverted index object that takes a JSON array of text objects and creates an index from the array. The index allows a user to search for text blocks in the array that contain a specified collection of words.

**USAGE**

-You can upload one or multiple files
-Ensure that your file is of type JSON
-After uploading, select a file in the drop-down list to invert
-Search query can be executed on each file or all files.
-Index can also be created for all files
